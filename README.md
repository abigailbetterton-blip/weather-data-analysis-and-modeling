# weather-data-analysis-and-modeling
Weather data analysis and modeling using regression, machine learning, and time series methods in Python

Overview

The goal of this project is to analyze relationships between meteorological variables and develop models to understand and predict weather-related outcomes.

Methods Used
Exploratory Data Analysis (EDA):
Investigated patterns and relationships between variables such as temperature, dew point, and visibility
Statistical Modeling:
Linear regression for continuous prediction
Logistic regression for classification tasks (e.g., rain vs. no rain)
Hypothesis testing (t-tests)
Time Series Analysis:
Decomposition into trend, seasonality, and residual components
Forecasting future values using time-series methods
Machine Learning Models:
Random Forest
Gradient Boosting
Model Evaluation

Models were evaluated using:

RMSE (regression performance)
AUC and accuracy (classification performance)
Tools and Technologies
Python
pandas, NumPy
matplotlib
scikit-learn
statsmodels
Key Takeaways
Demonstrated how different modeling approaches perform on real-world weather data
Highlighted the importance of data preprocessing and feature selection
Compared statistical and machine learning methods for predictive tasks
